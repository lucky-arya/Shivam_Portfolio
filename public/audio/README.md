# Audio Files for CRT TV Portfolio

This directory should contain the following audio files for the retro TV experience:

## Required Audio Files:

1. **tv-static.mp3** - Static noise sound (0.5-1 second)
2. **tv-click.mp3** - Button click sound (0.1-0.3 seconds)
3. **channel-flip.mp3** - Channel switching sound (0.3-0.5 seconds)
4. **power-on.mp3** - TV power on sound (1-2 seconds)
5. **power-off.mp3** - TV power off sound (0.5-1 second)
6. **ambient-hum.mp3** - Continuous ambient hum (looped, 2-5 seconds)

## Where to Find Audio:

You can find free retro TV sound effects from:
- **Freesound.org** - https://freesound.org/
- **Zapsplat.com** - https://www.zapsplat.com/
- **Mixkit.co** - https://mixkit.co/free-sound-effects/

Search terms:
- "TV static"
- "TV channel change"
- "TV power on"
- "Button click"
- "Electronic hum"
- "VHS noise"

## Temporary Solution:

If you don't have audio files yet, the app will still work - it just won't play sounds. The audio system is designed to fail gracefully.

## File Format:

- Format: MP3
- Sample Rate: 44.1kHz or 48kHz
- Bit Rate: 128-320 kbps
- Keep file sizes small (under 100KB each for best performance)

## Adding Your Audio:

Simply place the MP3 files in this directory with the exact names listed above.
